# Сборка APK

Этот проект содержит минимальный пример Android‑приложения на Java. Чтобы собрать APK:

1. Установите Android SDK и Java JDK.
2. Перейдите в каталог `android-java`.
3. Запустите скрипт сборки:
   ```bash
   ./gradlew assembleDebug
   ```
4. Готовый файл появится в `app/build/outputs/apk/debug/app-debug.apk`.
